# rbgit
Git command line interface in Ruby
## Installation
    gem build rbgit.gemspec
    
    gem install --local rbgit-0.0.1.gem
## Commands
Initialize rbgit

    rbgit init    

Check status

    rbgit status
    
Add files
  
    rbgit add file_name
    
Commit files

    rbgit commit -m "commit_message"
    
Log info

    rbgit log
    
Reset files/commits

    rbgit reset file_name/commit_id
    
Checkout files

    rbgit checkout file_names
